This is an **example standalone project**. A single Markdown file becomes one
project detail page, and one card on the Projects index.

## What the frontmatter does

- `tag` renders the category pill on the card and detail header.
- `featured: true` promotes it into the homepage "Selected Projects" list.
- `demo` and `source` become buttons — each shown only when its URL is present.
- Add an `image:` pointing at a file in this directory for a preview; without
  one, the striped placeholder is used.
